Algorithm HealthDataEntry
Input: UserHealthData[]  // array of daily health readings
Output: ConfirmationMessages[]

Set counter = 0
1. For each entry in UserHealthData do
2.     counter = counter + 1
3.     If entry.BloodSugar > 0 AND entry.HeartRate > 0 AND entry.BloodPressure != "" then
4.         Save entry to Database
5.         Append "Data Saved Successfully" to ConfirmationMessages
6.     Else
7.         Append "Invalid Data, Please Re-enter" to ConfirmationMessages
8.     End If
9. End For

10. Return ConfirmationMessages
End Algorithm
